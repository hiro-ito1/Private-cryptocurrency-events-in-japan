**バイバックとは？**  
プロジェクトが市場から自社トークンを買い戻す仕組み。  
株式の「自社株買い」に近く、2025年は注目度が急上昇。

**なぜ注目？**
- 収益をホルダーへ直接還元できる  
- トークン需給が改善し価格上昇につながる  
- 「収益力 × バイバック構造」が投資判断の基準に
