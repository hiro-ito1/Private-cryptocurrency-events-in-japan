📌 市場まとめ（2024〜2025）

2024年のBTC・ETH現物ETF承認により、暗号資産市場は
機関投資家が本格参入するフェーズ へ突入。

2025年10月は、

GOLD：連日の史上最高値更新

BTC：10万ドル割れ

という「安全資産 vs リスク資産」の構図が強まった。

11月は 日本の金融当局が一気に動いた歴史的タイミング となり、
法定通貨・銀行・株式・暗号資産の境界が溶ける“実装フェーズ”に入った。

特に金融庁による

暗号資産の金商法対象化

インサイダー規制の適用

国内銘柄の情報開示義務化

これは 日本のWeb3規制整備の最大級の転換点。

**バイバックとは？**  
プロジェクトが市場から自社トークンを買い戻す仕組み。  
株式の「自社株買い」に近く、2025年は注目度が急上昇。

**なぜ注目？**
- 収益をホルダーへ直接還元できる  
- トークン需給が改善し価格上昇につながる  
- 「収益力 × バイバック構造」が投資判断の基準に
